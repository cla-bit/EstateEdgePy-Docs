===========================================
Welcome to EstateEdgePy documentation!
===========================================

**EstateEdgePy** is a powerful and user-friendly Python package designed to enhance data analysis
of real estate properties data.
It simplifies the extraction and presentation of information for various properties in different regions,
offering enriched data display.


----------------------------------

**Prerequisites**

To get started with the EstateEdgePy library, ensure the following requirements are met:

* Python 3.10+

--------------

**Target Audience**

This library is intended for developers familiar with Python and real estate property data.

To get started, refer to the :doc:`get_started` section for installation instructions and requirements.
